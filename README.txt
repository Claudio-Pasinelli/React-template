    COME INSTALLARE IL PROGETTO

### Per scaricare le dipendenze del progetto, bisogna assicurarsi di essere nella cartella del progetto,
poi bisogna aprire il terminale bash e scrivere il comando "npm i" e poi premere il tasto invio ("Enter"),
ci potrebbero volere alcuni minuti perch√© l'installazione venga completata.



___________________________________________________________________________________________________________________________________________________________

NOTA BENE #################################################################################################################################################



Potrebbe accadere che un pacchetto - "react-icons" - non venga scaricato,
per sicurezza scrivere nel terminale "npm i react-icons" e aspettare per il completamento dell'installazione.

Se sciaguratamente invece ci dovessero essere altri pacchetti non scaricati o non trovati,
scrivere nel terminale "npm i <nome_pacchetto>" (come per "npm i react-icons").

I nomi dei pacchetti mancanti sono visibili nel terminale.

___________________________________________________________________________________________________________________________________________________________
   
    
    
    COME FAR PARTIRE IL PROGETTO

1) Per far partire il progetto bisogna scrivere, sempre da terminale, "npm run start".

2) Invece, per eseguire il processo del server, scrivere in un secondo terminale (sempre bash), "npm run server".
